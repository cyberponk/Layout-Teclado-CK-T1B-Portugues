# Layout-Teclado-CK-T1B-Portugues

Este layout serve para usar a versão Português do teclado CK-T1B no Windows 11 com todas as teclas mapeadas (e algumas extras).
O teclado originalmente tem um layout Inglês US modificado para incluir as teclas alternativas exclusivas deste teclado.
Foi criado um layout no Microsoft Keyboard Layout Creator (disponível publicamente).

COMO INSTALAR:
  * rodar o Setup e reiniciar o computador.
  * Abrir as configurações de idioma e região, editar as opções do idioma Português e adicionar o teclado Portuguese (Brazil) Jomaa CK-T1B.

COMO DESINSTALAR:
  * direto nos aplicativos instalados (configurações do Windows).
  * Reiniciar o computador para que tenha efeito.

COMO USAR:
  * Para usar as teclas alternativas, segure o ALT direito (ou CTRL+ALT esquerdo) e pressione a tecla desejada.
  * Nas teclas que possuem mais de uma alternativa, a de baixo é a normal e a de cima é segurando também o SHIFT.
  EXEMPLO:
    ALT direito + "+" =  »
    ALT direito + "+" + SHIFT =  «
  

Teclas extras:
Algumas teclas foram modificadas para se adaptar ao meu uso:
  * ALT DIREITO + 1 = ¹
  * ALT DIREITO + 1 + SHIFT = ½
  * ALT DIREITO + 2 = ⅓
  * ALT DIREITO + 2 + SHIFT = ⅓
  * ALT DIREITO + 3 = ³
  * ALT DIREITO + 3 + SHIFT = £
  * ALT DIREITO + [ = ±
  * ALT DIREITO + ? = •
  * ALT DIREITO + ? + SHIFT = ▹

COMO MODIFICAR:
  Basta abrir o arquivo do layout (.klc), editá-lo, depois criar um novo instalador, desinstalar o layout antigo, instalar o novo e reiniciar.
